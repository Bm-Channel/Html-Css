<Code>

É usada para indicar que o conteúdo dentro dela deve ser tratado como código e, geralmente, é exibido
com uma fonte monoespaçada (como Courier New)
para diferenciar do texto normal e facilitar a leitura do código.
altera a aparência do texto para uma fonte monoespaçada,
que é aquela onde cada caractere ocupa a mesma largura.
Isso ajuda a manter a formatação original do código, como espaços e quebras de linha,
que poderiam ser perdidos com a formatação padrão do texto.
Ex: input=('olá')

<Pre>

é usada para definir texto pré-formatado. Isso significa que o texto dentro dessa tag será exibido
exatamente como está escrito no código fonte, incluindo espaços em branco e quebras de linha. Em outras palavras,
o navegador não aplicará formatação padrão como a remoção de espaços extras ou quebras de linha, como faria com um parágrafo normal.
Ex: 

var nome = "João";let idade = 30; const PI = 3.14159;

Indentação

é usada principalmente para melhorar a legibilidade e a organização do código. Ela ajuda a visualizar a estrutura hierárquica dos elementos HTML,
tornando mais fácil a compreensão do código por desenvolvedores humanos e facilitando a manutenção e depuração.

<q>

é utilizada para marcar citações curtas em linha. Ela indica que o texto contido nela é uma citação,
e geralmente é renderizada com aspas pelo navegador,
mas sem quebrar a linha, como faria o elemento.
Ex: ele disse: Brasil não é para amadores

<blockquote>

é utilizado para indicar que um trecho de texto é uma citação de outra fonte,
geralmente um bloco maior de texto. Ele serve para destacar visualmente a citação,
separando-a do texto principal, não tem aspas automática.
Ex: Barbosa disse:

A pior ditadura é a do judiciário, contra ela não há a quem recorrer.

<blockquote cite =" ">

O atributo cite dentro de <blockquote> é usado para fornecer a URL da fonte da citação,
auxiliando na atribuição e fornecendo informações adicionais sobre a origem da citação.
Ex:

Idiota útil é uma vitima que também tem culpa

usada para indicar uma abreviação ou acrônimo dentro de um documento HTML. Ela serve para informar aos navegadores, leitores de tela e mecanismos de busca que o texto contido na tag representa uma abreviação, e opcionalmente, fornecer seu significado completo através do atributo title.

<bdo>

em HTML, que significa "Bidirectional Override", é usada para substituir a direção de exibição do texto. Em outras palavras, ela permite que você force a direção de um texto, ignorando a direção padrão definida pelo idioma ou contexto. Seus atributos são: ltr e rtl, ltr é esquerda para a direita e rtl é texto da direita para a esquerda, texto invertido.
Ex: carece carece
